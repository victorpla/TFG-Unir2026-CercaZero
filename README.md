# CercaZero 
**Plataforma web distribuida para el fomento de la economía circular vecinal.**

> **Contexto Académico:** Proyecto en desarrollo como Trabajo Fin de Grado (TFG) para el Curso de Adaptación al Grado en Ingeniería Informática de la UNIR.

---

##  Descripción General

CercaZero es una aplicación web responsive diseñada para reducir el desperdicio de recursos urbanos (como alimentos próximos a caducar u objetos en desuso) conectando a usuarios de un mismo vecindario. 

A través de un sistema basado en geolocalización, la plataforma facilita la donación y el intercambio gratuito de proximidad. El objetivo es eliminar las barreras logísticas tradicionales y fomentar la sostenibilidad local, asegurando al mismo tiempo la privacidad de la ubicación exacta de los donantes.

##  Funcionalidades Principales

* **Gestión de Usuarios:** Registro seguro y perfiles de usuario.
* **Inventario Geoespacial:** Publicación ágil de recursos excedentes con imágenes y detalles básicos.
* **Exploración por Proximidad:** Mapa interactivo y listado de ítems filtrados por distancia relativa al usuario.
* **Gestión de Intercambios:** Flujo de reserva y confirmación de entregas entre vecinos.
* **Privacidad por Diseño:** Sistema de ofuscación de coordenadas para proteger la dirección física real de los usuarios.

##  Stack Tecnológico Principal

El proyecto sigue una arquitectura Cliente-Servidor separada:

**Frontend (Aplicación Web Responsive)**
* **Tecnología Core:** React.js
* **Estilos:** Tailwind CSS
* **Mapas:** Integración con librerías de cartografía web (ej. Leaflet)

**Backend (API REST & Datos)**
* **Entorno del Servidor:** Node.js con Express
* **Base de Datos:** MongoDB (elegida por su capacidad de consultas espaciales nativas)
